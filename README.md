# Stock Briefing Dashboard - Frontend

Clean React frontend for the Stock Briefing tool. Connects to the backend running on Railway.

## Setup

1. Create new GitHub repo: `stock-briefing-frontend`
2. Upload all files
3. Deploy to Vercel with environment variable:
   - `REACT_APP_API_URL=https://your-railway-backend-url`

That's it. The dashboard will connect to your Railway backend and pull briefings.
